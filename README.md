# romance-cadastro

Projeto base para cadastro de consultoras da marca Romance.
Estrutura: frontend (Vite + React + Tailwind) e backend (Node + Express).

Como usar
1. Frontend
   - cd frontend
   - npm install
   - npm run dev

2. Backend
   - cd backend
   - npm install
   - node server.js
